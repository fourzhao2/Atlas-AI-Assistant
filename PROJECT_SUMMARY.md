## 构建和部署

### 构建命令

```bash
npm install          # 安装依赖
npm run build        # 构建扩展
npm run dev          # 开发模式
npm run type-check   # 类型检查
```

### 部署步骤

1. 生成图标（使用 `scripts/generate-icons.html`）
2. 运行 `npm run build`
3. 在浏览器中加载 `dist` 文件夹
4. 配置 API Key
5. 开始使用
