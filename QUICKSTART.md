# 🚀 Atlas 扩展 - 5 分钟快速入门

## 第一步：安装项目 (1 分钟)

```bash
# 克隆或下载项目
cd atlas-extension

# 安装依赖
npm install
```

## 第二步：构建扩展 (1 分钟)

```bash
# 构建
npm run build
```

构建完成后，`dist/` 文件夹中会包含扩展文件。

## 第三步：生成图标 (可选, 1 分钟)

1. 用浏览器打开 `scripts/generate-icons.html`
2. 点击每个图标下的"下载"按钮
3. 将下载的图标保存到 `public/icons/` 文件夹
4. 重新运行 `npm run build`

**或者跳过这步，扩展也能正常工作（只是没有图标）**

## 第四步：加载到浏览器 (1 分钟)

### Chrome:

1. 打开 `chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择项目中的 `dist` 文件夹

### Edge:

1. 打开 `edge://extensions/`
2. 开启"开发人员模式"
3. 点击"加载解压缩的扩展"
4. 选择项目中的 `dist` 文件夹

## 第五步：配置 API Key (1 分钟)

1. 点击浏览器工具栏中的扩展图标 (A)
2. 点击"设置"
3. 点击"AI 提供商"标签
4. 选择一个 AI 服务并输入 API Key：

### 获取 API Key：

**OpenAI (推荐新手)**
- 访问: https://platform.openai.com/api-keys
- 创建 API Key
- 复制并粘贴到扩展设置

**Anthropic Claude**
- 访问: https://console.anthropic.com/
- 获取 API Key

**Google Gemini**
- 访问: https://makersuite.google.com/app/apikey
- 生成 API Key

5. 点击"保存"

## 开始使用！🎉

### 方式 1: 总结网页

1. 访问任意网页（如维基百科）
2. 点击扩展图标
3. 点击"总结此页面"
4. 查看 AI 生成的摘要

### 方式 2: 打开侧边栏助手

1. 点击扩展图标
2. 点击"打开 AI 助手"
3. 在侧边栏中与 AI 对话
4. 尝试问："这个页面讲了什么？"

### 方式 3: 使用右键菜单

1. 在任意网页选中一段文字
2. 右键选择"解释选中内容"或"翻译选中内容"
3. AI 会在侧边栏给出回答

## 遇到问题？

### API Key 无效
- 检查是否正确复制了完整的 Key
- 确认 API 账户有余额

### 扩展无法加载
- 确保运行了 `npm run build`
- 检查是否启用了"开发者模式"
- 查看扩展页面的错误信息

### 某些页面无法使用
- Chrome 限制：`chrome://` 开头的页面无法使用扩展
- 这是浏览器的安全机制，属于正常现象
